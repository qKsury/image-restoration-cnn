# Image Restoration CNN

## 📌 Описание проекта

Проект посвящён восстановлению изображений с использованием сверточных нейронных сетей (CNN).  
Цель — научиться улучшать качество изображений низкого разрешения.

На текущем этапе реализован pipeline деградации изображений и датасет для обучения модели.

---

## ⚙️ Реализовано

### Деградация изображений
- JPEG-сжатие с настраиваемым качеством
- Размытие (Gaussian blur, defocus blur)
- Добавление шума
- Случайные сценарии деградации

### Датасет (`ImageRestorationDataset`)
- Онлайн генерация обучающих пар
- Деградация применяется ко всему изображению
- Случайный кроп (256×256)
- Возвращаемая пара: `(degraded, clean)`
- Фильтрация изображений по размеру
- Поддержка форматов: `.jpg`, `.jpeg`, `.png`

---

## 🔄 Как это работает

Для каждого изображения:

1. Загружается оригинальное изображение  
2. Применяется деградация ко всему изображению  
3. Генерируются случайные координаты кропа  
4. Вырезаются одинаковые участки из:
   - испорченного изображения  
   - оригинала  
5. Возвращается пара `(degraded, clean)`
